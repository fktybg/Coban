ÇOBAN — Türkçe Morfolojik Analiz Sistemi
Türkçe için geliştirilen kök + ek zinciri tabanlı morfolojik analiz sistemi.
ÇOBAN, Türkçe kelimeleri köklerine ve ek yapılarına ayırmayı hedefleyen deneysel bir NLP (Natural Language Processing) projesidir.

Özellikler
Türkçe kök analizi
Ek zinciri çözümleme
DFS tabanlı analiz sistemi
Büyük test havuzu desteği
Tamamen Python ile geliştirildi
Yerel ve bağımsız çalışma
Genişletilebilir mimari
Örnek Çıktılar
gözlükçülük
→ [GÖZ] [YPIsIs_lık] [YPIsIs_cı] [YPIsIs_lık]

sevmiştim
→ <K:sev> + <ÇDg> + <ÇHk> + <ÇK1>

Amaç
ÇOBAN’ın amacı Türkçe’nin eklemeli yapısını analiz edebilen,
hafif, anlaşılabilir ve geliştirilebilir bir morfolojik analiz sistemi oluşturmaktır.

Proje özellikleri:
Türkçe NLP
tokenizer geliştirme
spell checker
yerel yapay zekâ projeleri
LLM ön işleme sistemleri için altyapı oluşturmayı hedefler.

Teknolojiler
Python
DFS (Depth First Search)
Kök + ek zinciri yaklaşımı
